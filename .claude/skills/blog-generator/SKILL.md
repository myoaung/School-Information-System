# Blog Generator

1. Accept topic, outline, or keywords.
2. Generate blog structure:
   - Title (SEO-friendly)
   - Introduction with hook
   - Main sections with headings
   - Conclusion with CTA
   - Meta description
3. Write content:
   - Follow style guide tone
   - Include examples and code snippets
   - Add internal/external links
   - Optimize for readability
4. Generate frontmatter (title, date, tags, excerpt).
5. Export as Markdown or HTML.

# Slide Maker

1. Accept topic, outline, or document.
2. Generate slide structure:
   - Title slide
   - Agenda/overview
   - Content slides (one idea per slide)
   - Summary/conclusion
   - Q&A slide
3. Format for Marp:
   - Use Marp frontmatter
   - Apply theme (default, gaia, uncover)
   - Add speaker notes
   - Include images/diagrams
4. Export as:
   - Markdown (Marp format)
   - HTML presentation
   - PDF (via Marp CLI)

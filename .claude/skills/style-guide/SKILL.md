# Style Guide

1. Define tone rules:
   - **Formal** — professional, academic
   - **Casual** — friendly, conversational
   - **Technical** — precise, documentation-style
   - **Marketing** — persuasive, engaging
2. Define formatting rules:
   - Headings hierarchy (H1, H2, H3)
   - Sentence case vs title case
   - List formatting (bullets, numbers)
   - Code block styling
   - Link formatting
3. Define vocabulary:
   - Preferred terms
   - Avoided jargon
   - Abbreviations to use/avoid
4. Validate content against rules.
5. Suggest improvements for consistency.

# Study Notes Site

1. Accept source material (textbook, lectures, articles).
2. Organize into topics:
   - Main topics
   - Subtopics
   - Key concepts
   - Definitions
3. Generate notes:
   - Summary paragraphs
   - Bullet points
   - Code examples
   - Diagrams/tables
4. Create site structure:
   - Index page with navigation
   - Topic pages
   - Search functionality
5. Export as Astro site or static HTML.

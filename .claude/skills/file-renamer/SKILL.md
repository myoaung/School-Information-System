# File Renamer

1. Ask the user for the directory path and renaming pattern.
2. Support patterns:
   - **Prefix/Suffix** — add text before/after filename
   - **Replace** — find and replace text in filenames
   - **Numbering** — add sequential numbers (001, 002, etc.)
   - **Case** — convert to lowercase/uppercase/kebab-case/snake_case
   - **Date** — add date prefix/suffix
3. Preview changes before applying.
4. Apply renaming with confirmation.
5. Report how many files were renamed.

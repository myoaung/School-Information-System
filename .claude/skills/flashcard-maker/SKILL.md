# Flashcard Maker

1. Accept source material (notes, textbook, articles).
2. Extract key concepts:
   - Definitions
   - Questions and answers
   - Terms and explanations
   - Code snippets and outputs
3. Generate flashcards:
   - Front: Question or term
   - Back: Answer or definition
   - Tags for categorization
4. Format options:
   - Anki (importable format)
   - CSV for spreadsheet apps
   - Markdown for review
5. Include spaced repetition hints:
   - Difficulty level
   - Review frequency
   - Priority tags

# Deploy checklist

1. **Build** — Run `npm run build` and confirm no errors.
2. **Test** — Run `npm test` and confirm all tests pass.
3. **Lint** — Run `npm run lint` and fix any warnings.
4. **Publish** — Push to the deployment branch or run the deploy script.
5. **Verify** — Confirm the app is live and responding.

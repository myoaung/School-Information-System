# SchoolHub
## School Information System

A simple web app for students, teachers, and parents to stay connected.

**@myoaung**

---

# Problem

- Students miss announcements shared in different places
- Teachers waste time repeating notices
- Parents lack visibility into school updates

**Solution:** One platform for all school communication

---

# Features

✅ User registration and login with roles
✅ Announcements (teachers/admins create, students view)
✅ Class information display
✅ Contact form
✅ Responsive design (mobile + desktop)

---

# Stack

**Frontend:** React.js + Vite + Tailwind CSS
**Backend:** Node.js + Express.js + SQLite
**Auth:** JWT (JSON Web Tokens)

**Tools:** Skills, Agents, MCP servers

## Task 1: Database Migration

- `staff_contracts` table (id, staff_id, contract_type, start_date, end_date, salary, status, notes, created_at)
- Indexes on staff_id and end_date

## Task 2: API Routes

Create `server/routes/hr.js` with endpoints:
- GET /api/hr/staff - List all staff with contract info
- GET /api/hr/staff/:id - Get single staff member
- POST /api/hr/contracts - Create contract
- PUT /api/hr/contracts/:id - Update contract
- GET /api/hr/contracts/expiring - Get expiring contracts
- GET /api/hr/stats - HR statistics
